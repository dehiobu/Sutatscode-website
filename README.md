# Sutatscode Ltd Website

Professional website showcasing AI Adoption, Business Analytics, Data Resilience, and Cloud Solutions for NHS and Financial Services.

## Quick Deploy to Vercel (Recommended - 5 Minutes)

### Option 1: Deploy via Vercel Website (No coding required)

1. Go to [vercel.com](https://vercel.com) and sign up with GitHub
2. Click "Add New Project"
3. Import this repository
4. Vercel will automatically detect it's a Vite project
5. Click "Deploy"
6. Your site will be live at `https://your-project-name.vercel.app`

### Option 2: Deploy via Vercel CLI

```bash
# Install Vercel CLI
npm install -g vercel

# Deploy
cd sutatscode-website
vercel

# Follow the prompts and your site will be live!
```

## Deploy to Netlify (Alternative)

### Via Netlify Website

1. Go to [netlify.com](https://netlify.com) and sign up
2. Drag and drop the entire `sutatscode-website` folder
3. Your site will be live instantly!

### Via Netlify CLI

```bash
# Install Netlify CLI
npm install -g netlify-cli

# Deploy
cd sutatscode-website
netlify deploy --prod
```

## Local Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Custom Domain Setup

### On Vercel:
1. Go to your project settings
2. Click "Domains"
3. Add your custom domain (e.g., sutatscode.co.uk)
4. Follow DNS configuration instructions

### On Netlify:
1. Go to Site Settings → Domain Management
2. Add custom domain
3. Update your DNS records as instructed
